# Khabari

Khabari is a complete multi-page static news portal built with HTML5, CSS3, and vanilla JavaScript.

## Run

Open `index.html` in a browser. No build step, server, account, or extra asset download is required.

## Included

- 22 linked HTML pages
- 63 real-world, source-linked news briefs across World, India, Technology, AI, Business, Sports, Entertainment, Health, and Science
- Real photo URLs, local fallback SVG images, category banners, logo, favicon, and embedded media panels
- 3D card, hero, logo, and category-tile styling
- Responsive navigation, hero carousel, breaking ticker, live clock, dark mode, search, category filters, bookmarks, likes, sharing, reading progress, load more, toast notifications, newsletter validation, keyboard shortcuts, and scroll-to-top
- CSS selectors and pseudo-elements required by the project brief

## Keyboard Shortcuts

- `/` opens Search
- `d` toggles dark mode
- `t` scrolls to the top
- `b` bookmarks the current article page

## Editorial Note

Khabari uses original summaries with visible source links. It does not copy full copyrighted articles from publishers.
